Why do some stations say "no data available"?

The MTA doesn't provide arrival times for many train lines. Currently arrival data is only available for the 1, 2, 3, 4, 5, 6, and L trains.

Source Code

WTFT is open source and available on GitHub. It's designed to pull data from MtaSanitizer.
